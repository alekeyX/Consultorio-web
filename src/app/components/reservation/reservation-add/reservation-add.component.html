Registrar Historial Clínico

Bienvenido al sistema de reservas de citas médicas.
Pasos para habilitar el sistema:

  • Paso 1: Elija los dias hábiles para reservar citas.
  • Paso 2: Elija la fecha inicial para habilitar las reservas.
  • Paso 3: Elija la fecha limite de habilitación de reservas.
  • Paso 4: Elija el intervalo y horarios para las reservas.
  • Para escoger más de una opción, utilice (ctrl+click) ó (shift+click).
